# 희망일/시간 선택기 구현 계획

## 범위

주문 상세 모달에 `react-day-picker` 기반 희망일/시간 선택기를 추가한다.

## 작업 순서

1. `OrderDetail` 테스트에 픽업 주문의 날짜/시간 선택, 택배 주문의 날짜 전용 선택 시나리오를 추가한다.
2. `react-day-picker` 의존성을 추가한다.
3. 날짜/시간 선택 컴포넌트를 추가하고 `OrderDetail`의 희망일/픽업 시간 필드를 단일 선택 흐름으로 연결한다.
4. 상세 모달 CSS에 바텀시트, 빠른 날짜, 시간 칩, 적용/취소 액션 스타일을 추가한다.
5. 테스트, 빌드, 데스크톱/모바일 수동 검증을 수행한다.

## 검증

- `npm test -- src/components/OrderDetail.test.tsx`
- `npm test`
- `npm run build`
- 데스크톱과 모바일 폭에서 상세 모달을 열고 선택기가 넘치거나 겹치지 않는지 확인한다.
