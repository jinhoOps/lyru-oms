# 첫 입장 샘플과 주문 수집 채널 정리 설계

## 목표

처음 앱에 들어온 사용자가 빈 화면에서 당황하지 않도록 샘플 주문을 하나 보여주고, 주문 수집 영역의 채널 선택과 접힘 동작을 더 직관적으로 만든다.

## 결정

- 저장된 주문이 없을 때 앱 상태 초기화 과정에서 샘플 주문 1건을 넣는다.
- 샘플 주문은 고객명 `유리루`, 곶감말이 상품, 인스타그램 채널, 희망일과 수령 방식이 보이는 실제 주문 형태로 둔다.
- 주문 수집을 접으면 채널 선택, 원문 입력, 미리보기, 저장 버튼이 모두 함께 접힌다.
- `출처` 문구는 화면에서 `채널`로 바꾼다.
- 비밀번호가 이미 저장되어 있거나 새로 입력해 통과한 경우에도 앱 본문이 나오기 전에 1초 미만의 짧은 진입 애니메이션을 보여준다.

## 비목표

- 샘플 주문 삭제/복구 기능은 만들지 않는다.
- 채널 데이터 모델명 `source`는 내부 호환성을 위해 이번 작업에서 바꾸지 않는다.
