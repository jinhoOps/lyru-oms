# Static Pages Passcode Gate Design

## Goal

GitHub Pages처럼 공개 정적 호스팅에 올려도 우연한 접근을 줄이도록, 앱 진입 전에 간단한 패스코드 화면을 둔다.

## Design

- 보안 모델은 인증이 아니라 정적 웹용 접근 가림막이다. 공개 저장소에서는 코드를 보면 우회할 수 있다.
- 패스코드는 `9999`이며, 코드에는 원문 대신 SHA-256 해시를 저장한다.
- 사용자가 패스코드를 맞히면 `localStorage`에 `lyru-oms:access-granted` 값을 저장한다.
- 저장된 값이 있으면 다음 방문부터 바로 OMS 작업실을 보여준다.
- 잠금 화면은 기존 브랜드 톤을 유지하되, 첫 진입에서 부드러운 fade/blur/slide 애니메이션을 사용한다.
- 실패 시 입력값을 비우고 짧은 오류 문구와 흔들림 피드백을 제공한다.

## Components

- `src/components/AccessGate.tsx`
  - 잠금 상태 확인, 패스코드 입력, 해시 비교, 캐시 저장을 담당한다.
  - 성공 전에는 children을 렌더링하지 않는다.
- `src/App.tsx`
  - 기존 앱 화면을 `AccessGate`로 감싼다.
- `src/App.css`
  - 잠금 화면, 진입 애니메이션, 실패 피드백 스타일을 추가한다.

## Testing

- 캐시가 없으면 패스코드 화면이 보이고 기존 OMS 화면은 숨겨진다.
- `9999` 입력 후 제출하면 캐시가 저장되고 OMS 화면이 나타난다.
- 틀린 값은 오류를 보여주고 OMS 화면을 열지 않는다.
- 캐시가 있으면 패스코드 화면 없이 OMS 화면이 바로 보인다.
